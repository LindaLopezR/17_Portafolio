Proyecto: Grid System

Este proyecto consistió en usar la grid para la creación de páginas web, como una primera prueba, el uso no fue correcto pero esto ayudo a comprender como se debe utilizar para poder obtener lo mejor de esta.

Un proyecto díficil, que se realizó de manera como uno pensaba que era...

Proyecto: Article

En este proyecto se realizaron varias portadas de páginas web experimentando el acomodo de los elementos con la ayuda de la grid, con imágenes, texto, titulos, enlaces, etc.

Varios experimentos que hicieron que primero se planeara antes de codificar...

Proyecto: Chicama

Un proyecto rápido para demostrar las habilidades adquiridas con la grid, hace que el uso sea más fácil dandole así prioridad a que el acomodo sea más preciso y un estilo mucho mejor.

Una prueba más, esta vez para reafirmar lo que uno ha aprendido con respecto a los ejercicios pasados...

Proyecto: Perfil

A pesar de los conocimientos adquiridos, las diferentes formas de realizar una misma página web hace posible ver las posibilidades que se tiene y que uno eliga la que mayor le acomode como es mi caso.

Un ejercicio de mi, lo que importa es el contenido además de saber como lo codificas...
